ExpDate: Expiration Date Dataset

Paper
Dataset

Overview

ExpDate is a dataset for expiration date detection and recognition. It contains real-world expiration date images captured from food, beverage, and medical products. It also includes challenging cases with various date scales, different date fonts, reflection, complex backgrounds, etc. Figure 1 visualizes some images in ExpDate.



Details

ExpDate contains a total of 1767 images. We split it into training and test sets consisting of 1102 training and 665 test images, respectively. Additionally, the test set contains 1956 day, month, and year components, unlike the training set. ExpDate has near-horizontal expiration dates in the images. The width and height of the images vary in the range of [482, 2048] and [452, 2298] pixels, respectively. The dates are between the years 2018 and 2026. ExpDate has various types of expiration date fonts, including the dot-matrix font. It also includes images with multiple dates on the products (i.e., expiration and production dates). ExpDate includes 13 different expiration date formats. The expiration date formats are shown in Table 1.

Annotation

Each date in the images is annotated with class, bounding box coordinates, width and height of the image, and the date transcription. We utilize the top-left and bottom-right bounding box coordinates. The test set contains the annotations of the day, month, and year components in addition to date annotation, unlike the training set. Each component is annotated with class, bounding box, and transcription. The annotation format is shown in Figure 2.
